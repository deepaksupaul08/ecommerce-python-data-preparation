# E-Commerce Python Data Preparation

## Project Overview

This project demonstrates an end-to-end data preparation workflow using the Olist Brazilian E-Commerce dataset.

The objective was to transform multiple raw datasets into a single analytics-ready dataset through data profiling, data quality assessment, cleaning, transformation, feature engineering, and data integration using Python and Pandas.

The final dataset is designed to support business reporting and interactive dashboards in Power BI.
